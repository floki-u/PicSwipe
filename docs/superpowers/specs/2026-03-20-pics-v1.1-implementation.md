## 三、关键技术决策

### 3.1 照片滑动卡死问题
- **根因**：`HapticService.thresholdReached()` 在 `onChanged` 每帧触发，创建新 UIImpactFeedbackGenerator 60 次/秒
- **方案**：`hasTriggeredHaptic` 布尔防抖，越过阈值仅触发一次

### 3.2 PhotoKit 主线程阻塞
- **根因**：`PHAsset.fetchAssets()` + `enumerateObjects()` + `getFileSize()` 全在主线程
- **方案**：`Task.detached(priority: .userInitiated)` 移到后台线程

### 3.3 视频播放无声音
- **根因**：iOS 要求显式配置 `AVAudioSession.category = .playback`
- **方案**：在 `VideoPlayerManager.loadVideo()` 开头设置音频会话

### 3.4 动画时序冲突
- **根因**：`AnimatedArrowView` 在 `.onAppear` 中立即启动 `repeatForever` 动画与首帧布局冲突
- **方案**：延迟 0.1s 后通过 `withAnimation` 启动

### 3.5 "再来一组"导航
- **方案**：先 `path.removeLast(path.count)` 回到首页，延迟 0.1s 后 `path.append(.swipe(mode))` 直接进入新会话
