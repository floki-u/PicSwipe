## 一、设计语言

### 品牌标识
- **应用名称**：PicS（用户可见）/ PicSwipe（代码标识）
- **品牌色调**：Fresh Mint — 清新薄荷绿
- **情绪**：轻松、快速、干净
- **图标**：深色底 + 品牌渐变圆 + 粗体 "S" + "PicS" 字标

### 色彩系统

| Token | 色值 | 用途 |
|-------|------|------|
| `brandPrimary` | `#43E97B` | 保留/成功/主按钮 |
| `brandSecondary` | `#38F9D7` | 渐变终点、辅助强调 |
| `destructiveRed` | `#FF453A` | 删除/警告 |
| `warningYellow` | `#F4C542` | 存储警告 |
| `appBackground` | `#111111` | 全局深色背景 |
| `surfaceBackground` | `white 6%` | 卡片/区块背景 |
| `textSecondary` | `#888888` | 次要文字 |
| `textMuted` | `#555555` | 辅助文字 |

### 间距系统

| Token | 值 | 用途 |
|-------|-----|------|
| xs | 4pt | 紧凑间距 |
| sm | 8pt | 小间距 |
| md | 16pt | 标准间距 |
| lg | 24pt | 大间距 |
| xl | 32pt | 特大间距 |
| pagePadding | 16pt | 页面左右边距 |

### 圆角系统

| Token | 值 | 用途 |
|-------|-----|------|
| hero | 22pt | 英雄区卡片 |
| card | 16pt | 普通卡片 |
| button | 14pt | 按钮 |
| thumbnail | 8pt | 缩略图 |
| chip | 12pt | 标签/胶囊 |

### 2.2 照片滑动页 SwipeView (Photo Mode)

```
┌─────────────────────────────────┐
│ [◀]        🗑 3        2/20    │  ← 半透明渐变顶栏
│─────────────────────────────────│
│                                 │
│                                 │
│         ┌───────────┐           │
│         │           │           │
│  ▓▓▓▓▓  │   照 片   │           │  ← 左滑时左侧红色渐变
│  红色渐变 │   内 容   │           │
│         │           │           │
│         │           │  ▓▓▓▓▓   │
│         └───────────┘  绿色渐变  │  ← 上滑时顶部绿色渐变
│                                 │
│           2024/06/18            │  ← 日期（纯净模式可隐藏）
│            3.2 MB               │
│ ████████████░░░░░░░░░░░░░░░ 3pt │  ← 进度条
└─────────────────────────────────┘
```

**手势交互：**

| 手势 | 动作 | 视觉反馈 | 飞出方向 |
|------|------|---------|---------|
| ↑ 上滑 | 保留 | 顶部绿色渐变光晕 | 沿拖拽方向向上飞出 |
| ← 左滑 | 删除 | 左侧红色渐变光晕 | 沿拖拽方向向左飞出 |
| ↓ 下滑 | 回看上一张 | 底部白色渐变光晕 | 弹回 |
| 点击 | 切换纯净模式 | 顶栏/底栏/进度条隐藏 | — |
| 双指缩放 | 查看细节 | 1x-4x 缩放 | — |
| 快速轻甩 | 同方向动作 | 速度检测触发 | 同上 |

**动画参数：**
- 跟手衰减系数：0.5（手指移 100pt → 卡片移 50pt）
- 阈值：屏幕尺寸 / 6
- 飞出：`.easeOut(duration: 0.3)`
- 回弹：`.spring(response: 0.35, dampingFraction: 0.75)`
- 旋转：±5° 基于水平位移
- 触觉：越过阈值触发一次（防抖）

### 2.3 视频滑动页 SwipeView (Video Mode)

```
┌─────────────────────────────────┐
│ [◀]        🗑 2        1/10    │
│─────────────────────────────────│
│                                 │
│                                 │
│                                 │
│         ┌───────────┐    ┌──┐  │
│         │           │    │🗑│  │  ← 右侧删除按钮（红色）
│         │  视 频    │    └──┘  │
│         │  播 放    │    ┌──┐  │
│         │  内 容    │    │↩│  │  ← 右侧撤回按钮（有标记时）
│         │           │    └──┘  │
│         │     ▶     │          │  ← 暂停时显示播放图标
│         └───────────┘          │
│                                 │
│  ──●────────────────────── 进度  │  ← 可拖拽进度条
│  0:15 / 1:23            🔊     │  ← 时间 + 静音切换
│ ████████░░░░░░░░░░░░░░░░░░ 3pt │
└─────────────────────────────────┘
```

**手势交互：**

| 手势 | 动作 | 说明 |
|------|------|------|
| ↑ 上滑 | 下一个视频 | 仅垂直方向，水平锁定为 0 |
| ↓ 下滑 | 上一个视频 | 同上 |
| ← → 左右滑 | 无响应 | 锁定水平 |
| 点击视频 | 暂停/继续 | 暂停时中央显示 ▶ 图标 |
| 长按 0.3s | 2x 加速 | 松手恢复 1x，左上角 "2x ⏩" 标签 |
| 拖拽进度条 | Seek | 实时更新播放位置 |

**右侧浮动按钮：**
- 🗑 删除：红色圆形，点击标记删除并自动跳到下一个（左飞出动画）
- ↩ 撤回：半透明圆形，仅在有标记时出现（scale+opacity 动画）
- 位置：右侧垂直居中

